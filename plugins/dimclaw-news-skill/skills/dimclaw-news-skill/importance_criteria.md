# 新闻重要性判断标准

对所有来源统一适用，决定一条新闻进入 **main**（重要新闻，附摘要和洞察）还是 **brief**（简要新闻，仅标题和链接）。

**核心原则：main 放"需要知道"的，brief 放"可以知道"的。**

## 进 main 的条件（满足任一即可）

**按事件影响力：**
- 影响大量人群的政策/监管变动（央行决议、反垄断、签证政策等）
- 头部公司（FAANG、BAT 级别）的重大战略动作（不是小功能更新）
- 新技术突破 / 新 SOTA / 重要开源项目发布
- 突发事件（地缘冲突、自然灾害、重大安全漏洞等）
- 金融市场重大波动（单日大涨大跌、重要经济数据异常等）

**按受众相关性（读者背景：{YOUR_BACKGROUND}）：**
- 与读者核心兴趣领域高度相关：{YOUR_INTERESTS}
- {YOUR_DOMAIN_1}
- {YOUR_DOMAIN_2}
- {YOUR_DOMAIN_3}

## 进 brief 的（不满足上述条件的）

- 常规融资新闻（除非金额巨大或公司极知名）
- 产品小更新 / 版本迭代
- 娱乐/八卦/体育类热搜
- 纯观点/评论文章（除非观点本身引发重大争议）
- 工具类小项目（GitHub Trending 上的小工具）
- 例行数据发布（月度统计、常规报告等）

## 量化指标作为辅助参考

各来源的热度数据（HN points、微博热度值等）不作为决定性标准，但在内容语义判断接近时，热度高的更倾向进 main。
